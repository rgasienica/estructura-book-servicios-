# Empresa Habitat

Sistema de diseño y guía de estilo para Habitat & Decor.

## Qué encontrar aquís

- **Guía de estilo**: Colores, tipografía, componentes, iconos
- **Tokens de diseño**: Variables CSS listas para usar
- **Plantillas**: Plantillas HTML listas
- **Snippets**: Bloques reutilizables

## Uso rápido

```html
<link rel="stylesheet" href="theme/css/tokens.css">
```
